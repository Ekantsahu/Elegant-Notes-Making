## 📌 Introduction
This is a simple Notes Making App built using HTML, CSS, and JavaScript.  
It allows users to create, edit, and delete notes directly in the browser.  
The goal of this project is to practice DOM manipulation and build a clean, interactive UI.

---

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript  

---

## ✨ Features
- Create new notes  
- Search existing notes  
- Delete notes  
- Auto-save notes in local storage  
- Clean and responsive UI  

---

## ⌨️ Keyboard Shortcuts
- `Enter` → Create new note  
- `Backspace` → Delete empty note   

---

## 🧠 Development Process
- Started with basic HTML structure  
- Styled the UI using CSS for better user experience  
- Used JavaScript for DOM manipulation  
- Implemented local storage to save notes  
- Focused on making the app simple and user-friendly
